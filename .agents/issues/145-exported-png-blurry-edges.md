# Issue #145: Exported PNG image edges are blurry

**GitHub:** https://github.com/vinothpandian/react-sketch-canvas/issues/145
**Created:** 2024-02-02
**Labels:** None
**Type:** Bug

## Description

When exporting drawings as PNG, the edges of strokes appear unclear and blurry.

## Steps to Reproduce

1. Draw something on the canvas
2. Export as PNG
3. Observe that stroke edges are blurry

## Expected Behavior

Exported PNG should have clear, crisp edges matching what is displayed on the canvas.

## Environment

- OS: macOS
- Browser: Chrome
- Library Version: 6.2.0

## Evidence

Screenshots in the GitHub issue show the difference between blurry exported edges and the expected clear edges.
