# Issue #25: Is there a way to rotate the canvas?

**GitHub:** https://github.com/vinothpandian/react-sketch-canvas/issues/25
**Created:** 2021-07-16
**Labels:** enhancement, help wanted, good first issue
**Type:** Feature Request

## Description

Request for the ability to rotate the canvas, particularly for image annotation use cases where the background image may need rotation.

## Use Case

The reporter uses the module to annotate images by:
1. Setting an image as the background
2. Using the canvas as a layer on top for annotations

When attempting to rotate the canvas, the background image rotates but the pen behavior becomes incorrect - the pen moves in different directions than expected.

## Desired Behavior

Support for rotating the canvas (and background image) while maintaining correct pen/drawing behavior relative to the rotated view.

Video demonstration available: https://www.loom.com/share/28effbb145bf4fd88af7bc4f2bc60d52
