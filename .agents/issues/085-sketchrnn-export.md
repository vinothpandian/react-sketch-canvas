# Issue #85: Add export to SketchRNN format

**GitHub:** https://github.com/vinothpandian/react-sketch-canvas/issues/85
**Created:** 2022-09-19
**Labels:** None
**Type:** Feature Request

## Description

Request to add support for exporting drawings in SketchRNN format.

## Context

SketchRNN is a recurrent neural network model for generating vector drawings. The format is commonly used in machine learning applications for sketch-based tasks.

A SketchRNN export would enable users to:
- Use drawings for ML training data
- Integrate with sketch-based ML applications
- Export drawings in a format compatible with the Quick, Draw! dataset
