# Issue #121: Canvas vertical stroke not working in LINE web browser

**GitHub:** https://github.com/vinothpandian/react-sketch-canvas/issues/121
**Created:** 2023-06-08
**Labels:** None
**Type:** Bug / Browser Compatibility

## Description

Vertical strokes on the canvas do not work correctly in the LINE application's built-in web browser on iPhones. The screen moves instead of drawing strokes, and only dots or small lines can be created.

## Steps to Reproduce

1. Go to the LINE application
2. Paste the URL where the canvas is deployed
3. Click on the URL (opens in LINE's web browser)
4. Attempt to draw vertical strokes on the canvas

## Actual Behavior

The screen scrolls/moves instead of displaying the strokes. Only dots or small lines can be drawn.

## Expected Behavior

Vertical strokes should be drawn properly and precisely.

## Environment

- Device: iPhones (all versions)
- Browser: LINE web browser

## Evidence

Video demonstrations available in the GitHub issue.
