# Issue #171: Writing on the drawing board with the iPad pen will lose focus and a selection box will pop up

**GitHub:** https://github.com/vinothpandian/react-sketch-canvas/issues/171
**Created:** 2024-09-03
**Labels:** None
**Type:** Bug

## Description

When writing on the drawing board with an iPad pen (Apple Pencil), the canvas loses focus and a browser selection box appears. This interrupts the drawing experience.

## Expected Behavior

The browser selection behavior should not be triggered during drawing with a stylus.

## Environment

- OS: iOS
- Browser: Safari
