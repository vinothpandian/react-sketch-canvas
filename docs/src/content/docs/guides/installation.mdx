`react-sketch-canvas` is a single npm package. It renders an SVG drawing surface, so there is no extra canvas runtime or browser plugin to configure.

## Install

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab value="npm">
    ```sh
    npm install react-sketch-canvas
    ```
  </Tab>
  <Tab value="yarn">
    ```sh
    yarn add react-sketch-canvas
    ```
  </Tab>
  <Tab value="pnpm">
    ```sh
    pnpm add react-sketch-canvas
    ```
  </Tab>
</Tabs>

## Quick start

<Steps>
  <Step>
    ### Import the component

    ```tsx twoslash
    import { ReactSketchCanvas } from "react-sketch-canvas";
    ```
  </Step>
  <Step>
    ### Render it with the props you care about

    Width, height, stroke color, and canvas color are the most common starting props.

    ```tsx twoslash title="App.tsx"
    import React from "react";
    import { ReactSketchCanvas } from "react-sketch-canvas";

    // ---cut---
    export function App() {
      return (
        <ReactSketchCanvas
          width="100%" // [!code highlight]
          height="150px" // [!code highlight]
          canvasColor="transparent" // [!code highlight]
          strokeColor="#a855f7" // [!code highlight]
        />
      );
    }
    ```
  </Step>
  <Step>
    ### Try it

    <ExampleBlock source={BasicExampleSource}>
      <BasicExample />
    </ExampleBlock>
  </Step>
</Steps>
