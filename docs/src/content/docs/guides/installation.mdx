Install `react-sketch-canvas` in the same package where your React components live. The component renders an SVG drawing surface, so there is no extra canvas runtime or browser plugin to configure.

import { Tabs, TabItem } from '@astrojs/starlight/components';

<Tabs>
  <TabItem label="npm">
    ```sh title="Installing dependencies…"
    npm install react-sketch-canvas
    ```
  </TabItem>
  <TabItem label="yarn">
    ```sh title="Installing dependencies…"
    yarn add react-sketch-canvas
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh title="Installing dependencies…"
    pnpm add react-sketch-canvas
    ```
  </TabItem>
</Tabs>


## Quick Start

Start with `ReactSketchCanvas` as a regular controlled-looking React component. You choose the dimensions, colors, and stroke width through props, then layer your own UI around it when you need buttons, sliders, or export actions.

The example below keeps the setup intentionally small: render the canvas, give it a visible border, and let users draw immediately.

import ExampleBlock from "../../../components/ExampleBlock.astro";
import BasicExample from "../../../examples/BasicExample";
import BasicExampleSource from "../../../examples/BasicExample.tsx?raw";

<ExampleBlock source={BasicExampleSource}>
  <BasicExample client:load />
</ExampleBlock>
