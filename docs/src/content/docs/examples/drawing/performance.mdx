React Sketch Canvas is built to stay usable with dense SVG drawings. Use this demo to add up to 1,000 strokes at a time, with up to 1,000 random points per stroke, and watch the live canvas keep up as the total grows.

Try pen strokes for heavy path rendering, then add eraser masks to exercise the masking path too. The counter shows the total strokes and points currently held by the canvas.
