Background images are useful when the drawing is an annotation layer: mark up a photo, trace over a template, or collect notes on top of an existing asset.

Use `backgroundImage` with `preserveBackgroundImageAspectRatio` to draw over an image while controlling how the image fits the SVG canvas. The background stays separate from the user's strokes, so you can preserve the source image and export the combined result when needed.
