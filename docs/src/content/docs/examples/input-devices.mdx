## Overview

By default, `ReactSketchCanvas` accepts every pointer type the browser reports. That is the right default for simple drawing surfaces, but production interfaces often need more control.

Use `allowOnlyPointerType` when the input device matters. For example, a stylus-first note app may want to ignore finger touches so users can rest their hand on a tablet without leaving marks.

These are the available options:
- `all` (default) - All input devices are allowed.
- `pen` - Only pen input is allowed. This includes stylus, Apple Pencil, Wacom, Windows Surface Pen, and other pen devices.
- `mouse` - Only mouse input is allowed.
- `touch` - Only touch input is allowed. This includes finger touch and other touch devices.
