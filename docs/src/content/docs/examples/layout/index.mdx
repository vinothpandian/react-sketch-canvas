`ReactSketchCanvas` reads pointer events in viewport-relative coordinates and subtracts the canvas's own `getBoundingClientRect()`. Both values move together when the page, an ancestor element, or a CSS transform changes position, so strokes always land where the pointer is — no matter how the canvas is laid out.

These examples exercise the three common layout edge cases that used to trip up coordinate math.
