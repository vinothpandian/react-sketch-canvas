Some products need more than the final drawing. You may also need to know how long the user actively spent sketching, signing, or annotating.

Enable `withTimestamp` before calling `getSketchingTime()`. With timestamps attached to each stroke, the canvas can calculate active drawing time without counting idle moments between interactions.

This is useful for analytics, review workflows, and experiences where completion time is part of the captured data.
